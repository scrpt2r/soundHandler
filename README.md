# soundHandler

A modular audio framework built for Roblox developers.

## Structure

```
src/
  ├─ __index.luau    ← entry point
  ├─ __main__.luau   ← constructor
    ├─ libs/
```

## Setup

```bash
aftman install
rojo serve
```

## Usage

```lua
local soundHandler = require(
    game.ReplicatedStorage.soundHandler
)
```
